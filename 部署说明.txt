WebSSH 一键部署命令
====================

1. 一键部署（国内加速）：

bash <(curl -sL https://ghfast.top/https://raw.githubusercontent.com/mubaiqq/webssh-terminal/main/deploy.sh)

2. 手动部署：

git clone https://github.com/mubaiqq/webssh-terminal.git
cd webssh-terminal
bash deploy.sh

3. 完成后自动配置：
   - 本地访问: http://localhost:9111
   - 公网隧道: https://xxx.serveousercontent.com（可选）
   - pm2 进程守护（崩溃自动重启）
   - 开机自启（systemd）

4. 管理命令：

   pm2 status          # 查看状态
   pm2 logs            # 查看日志
   pm2 restart         # 重启
   bash stop.sh        # 停止所有

5. 卸载：

   bash uninstall.sh

注意事项：
- 需要 Node.js >= 16
- 需要 npm
- 需要 ssh 客户端（用于 serveo 隧道，可选）
- 数据保存在 data/ 目录，重启不丢失
- 默认密码 123456，登录后请修改
