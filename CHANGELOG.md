# 更新日志

## v1.2.0 - 2026-04-23

### 🐛 Bug 修复
- **终端滚动修复**：移除 `will-change: transform`，修复 xterm viewport 溢出问题
- **终端高度约束**：用固定像素（150px 底部留白）控制终端高度，不再依赖百分比或 CSS 约束
- **智能自动滚动**：检测用户滚动位置，手动上翻时停止自动滚动，输入时恢复跟随
- **页面嵌套修复**：修复多余的 `</div>` 导致命令和笔记页面被嵌套在终端页面内无法显示的问题
- **列表排序**：服务器、命令、笔记列表按创建时间倒序排列，最新的在最上面

### ✨ 新功能
- **数据导入导出**：设置面板新增数据备份功能，支持导出 JSON 备份文件和导入恢复
- **更新日志**：新增 `CHANGELOG.md` 文件

### 🔧 部署改进
- **deploy.sh 重写**：支持从仓库目录直接部署（rsync 同步，排除 data/ 和 .git/）
- **端口冲突自动处理**：启动前自动杀掉占用端口的进程
- **数据保留**：更新时自动备份 data/ 目录，代码重置后恢复

## v1.1.0 - 2026-04-23

### ✨ 新功能
- **虚拟键盘**：悬浮可拖拽虚拟键盘，支持快捷键/数字/字母/符号/F键 5 页翻页
- **粘贴页面**：虚拟键盘新增粘贴页面，支持从剪贴板粘贴内容并发送到终端
- **子目录反代**：支持通过 `BASE_PATH` 环境变量部署到子目录

### 🔧 改进
- **主题切换**：暗色/亮色主题切换
- **背景自定义**：支持纯色/外链/本地上传背景图片
- **终端字体大小**：设置面板可调整终端字号

## v1.0.0 - 初始版本

### ✨ 核心功能
- 🔐 密码认证（Cookie 记忆 6 小时）
- 🖥️ 多标签终端（xterm.js + WebSocket + node-pty）
- 📋 服务器管理（添加/编辑/删除，支持密码和密钥认证）
- ⚡ 快捷命令（保存常用命令，一键发送到终端）
- 📝 笔记（内置记事本，自动保存）
- 🔒 pm2 进程守护 + systemd 开机自启
- 📱 移动端适配
